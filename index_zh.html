關於我

在過去的時間裡，我曾任職於HR，負責員工學習與福委管理工作。這份工作讓我學會了如何有效溝通、協作，以及處理複雜的問題。然而，在這段時間裡，我發現自己對技術越來越有興趣，特別是如何透過程式設計解決現實中的挑戰。

帶著對新領域的好奇心，我決定轉換跑道，開始學習網頁開發。我目前在溫哥華進修，專注於前端技術，如 HTML、CSS 和 JavaScript，同時也在逐步掌握 Git 和 API 的應用。儘管我還在學習階段，但我每天都能感受到自己在成長，尤其是當我完成一個又一個練習專案時，那種成就感讓我確信自己選擇了正確的方向。

我的目標是成為一名全端開發者，並將人資背景中學到的溝通與問題解決能力應用於開發工作。我相信這段從人資到程式開發的轉型，將賦予我獨特的視角與競爭力，幫助我在科技領域找到屬於我的位置。
